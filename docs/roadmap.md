# Roadmap

RepoPulse is an early-stage maintainer tool. The roadmap focuses on making repository health checks more useful in real open source workflows.

## Near Term

- Publish installation-ready releases.
- Expand check coverage for common Python, JavaScript, and documentation patterns.
- Add richer Markdown reports for maintainers and contributors.
- Improve JSON output stability for automation.
- Add examples for running RepoPulse in GitHub Actions.

## Maintainer Automation

- Generate issue-ready recommendations from failed checks.
- Support configurable score weights per project.
- Add comparison reports across releases or branches.
- Provide templates for release readiness checks.

## Security and Quality

- Keep sensitive file detection content-safe by reporting file names only.
- Expand security baseline checks around Dependabot, CodeQL, and security policy setup.
- Document responsible usage for private repositories and tokens.

## Community

- Keep contributor setup lightweight.
- Label beginner-friendly issues.
- Use public issues for roadmap discussion and private advisories for vulnerabilities.
